# Personal Photo Instructions

## Save Your Photo Here

1. **Save your personal photo** as `profile.jpg` in this directory
2. **Ensure the filename** is exactly `profile.jpg` (lowercase)
3. **Recommended dimensions**: 800x1000px or similar 4:5 aspect ratio
4. **File format**: JPG, PNG, or WebP
5. **File size**: Under 2MB for optimal performance

## Current Usage

Your photo will be displayed in:

- **Hero Section**: Main homepage image
- **About Section**: Profile image

## Image Effects

- Grayscale filter (removes on hover)
- Rounded corners and shadows
- Responsive scaling
- Dark/light mode compatible
